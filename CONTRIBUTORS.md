## How to Contribute

We welcome contributions of all kinds:

- **Bug Reports** — Found something broken? [Open an issue](https://github.com/streamthoughts/jikkou/issues/new)
- **Feature Requests** — Have an idea? [Start a discussion](https://github.com/streamthoughts/jikkou/issues/new)
- **Code** — Fix a bug or add a feature. See [CONTRIBUTING.md](./CONTRIBUTING.md) for guidelines
- **Documentation** — Improve docs, fix typos, add examples
- **Testing** — Write tests, report edge cases

All contributors are expected to follow the project's [Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Recognition

Contributors are automatically featured on the [README](./README.md) and on the
[GitHub contributors page](https://github.com/streamthoughts/jikkou/graphs/contributors).
